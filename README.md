#  Calculator App

A simple calculator application built using **.NET Framework 4.8** and **Windows Forms** in **Visual Studio 2017**.  
This project was created as part of a degree assignment to demonstrate basic Windows Forms development and arithmetic operations in C#.

---

##  Objective

The main objective of this project is to create a simple and user-friendly calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division using Windows Forms in .NET.

---

##  Features

- Simple and intuitive user interface  
- Supports basic arithmetic operations:
  - Addition  
  - Subtraction  
  - Multiplication  
  - Division  
- Displays input and result in a text box  
- Handles invalid input gracefully  
- Clear (DEL) and equals (=) functionality  

---

##  Tools & Technologies Used

- **IDE:** Visual Studio 2017  
- **Framework:** .NET Framework 4.8  
- **Language:** C#  
- **UI Framework:** Windows Forms (WinForms)

---

##  How to Run the Application

1. **Clone the repository**
```bash
git clone https://github.com/DanushkaMadush/DanushkaMadushan_E193919_Calculator_App.git
```
2.**Open the Project**
```bash
Navigate to the project folder and open **.sln** file using Visual Studio
```
3.**Build and Run**
```bash
Press Start or use shortcut F5 to build and run the calculator.
```
